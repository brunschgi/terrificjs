Class Tc.Module

Base class for the different modules.

Constructor

Tc.Module ( )

Properties

$ctx - jQuery

Contains the module context.

connectors - Array

Contains the attached connectors.

dependencyCounter - Object

Contains the dependency counter for the different phases.

modId - String

Contains the unique module ID.

onBinding - object

The following counters have to be at least zero, so that the onBinding callback is loaded as a dependency for onBinding and the onBinding phase is completed for afterBinding.

sandbox - Sandbox

The sandbox to get the resources from.

Class Tc.Application

Responsible for application-wide issues such as the creation of modules.

Constructor

Tc.Application ( )

Properties

$ctx - jQuery

The jQuery context.

classes A module can have three types of classes: - object

config - Object

The configuration.

connectors - Object

Contains references to all connectors on the page.

modules - Array

Contains references to all modules on the page. Could be useful for example when there are interactions between Flash <-> JS.

sandbox - Sandbox

The sandbox to get the resources from This sandbox is shared between all modules.

wildcardComponents - Array

Contains references to all wildcard components on the page.
